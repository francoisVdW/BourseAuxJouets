Aide à propos des caisses

Pour chaque bourse, on défini un nombre de caisses (de 1 à ...).
Ce système permet de mieux suivre les opérations de ventes.
Quand on se connecte au serveur (intraNet) bourse, la page d'acceuil montre l'état des caisses.
Pour chaque caisse, un synoptique indique son état :
Cette caisse est fermée. On peut utiliser cette caisse pour enregistrer les ventes.
Cette caisse est occupé (verouillée). Un autre participant utilise cette caisse actuellement, si vous tentez d'utiliser cette caisse, la connexion sera refusée.
Vous êtes en train d'utiliser cette caisse ! Elle est verouillée pour les autres participants
Pour déverouiller cette caisse c'est à dire la rendre "Prêt pour l'utilisaton" pour les autres participants, il suffit d'ouvrir (re-ouvrir) cette caisse puis de retourner au menu principal.
